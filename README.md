# AI Storyboard Prompts

English | [简体中文](README.zh-CN.md)

A bilingual prompt repository for AI storyboard creation, character design, scene generation, camera movement, and script-to-video workflows.

This repository is designed for creators who want reusable prompts for:

- character design
- scene concept generation
- storyboard planning
- camera movement design
- short drama visual workflows
