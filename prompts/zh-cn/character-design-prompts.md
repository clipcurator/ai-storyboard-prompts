# Character Design Prompts

## Prompt 1：角色设计表

```text
请为一个短剧角色生成角色设计表。

角色信息：
- 姓名：
- 年龄：
- 身份：
- 性格：
- 核心动机：
- 与其他角色关系：
- 视觉风格：[写实 / 日漫 / 3D]

请输出：
1. 基础形象
2. 面部特征
3. 服装设计
4. 色彩风格
5. 三套服装变化
6. 后续场景一致性注意事项
```

## Prompt 2：多角度角色视图

```text
请为这个角色生成多角度角色提示词：

- 正面
- 侧面
- 四分之三角度
- 特写

要求所有视图保持同一身份、服装和情绪基调。
```
