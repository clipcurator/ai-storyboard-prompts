# Storyboard Prompts

## Prompt 1: Script to storyboard

```text
Convert the following script scene into a storyboard shot list.

For each shot, provide:
- shot number
- shot type
- subject
- action
- camera angle
- camera movement
- visual prompt
- duration estimate

Script scene:
[paste scene]
```

## Prompt 2: Story beat storyboard

```text
Create a storyboard for this story beat:

Story beat:
[describe beat]

Goal:
- preserve narrative clarity
- show emotional progression
- avoid unnecessary shots
```
