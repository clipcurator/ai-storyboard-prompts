# Scene Design Prompts

## Prompt 1: Scene concept art

```text
Create a scene concept prompt for a short drama.

Scene context:
- location:
- time of day:
- emotional tone:
- characters present:
- key action:
- visual style: [realistic / anime / 3D]

Output:
1. wide shot prompt
2. close-up detail prompt
3. overhead view prompt
4. lighting notes
5. continuity notes
```

## Prompt 2: Scene consistency

```text
Create a scene consistency guide for this location.

Include:
- fixed layout
- recurring props
- lighting rules
- color palette
- camera-safe angles
```
