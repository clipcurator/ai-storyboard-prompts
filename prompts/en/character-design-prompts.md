# Character Design Prompts

## Prompt 1: Character sheet

```text
Create a character design sheet for a short drama character.

Character context:
- name:
- age:
- role:
- personality:
- motivation:
- relationship to other characters:
- visual style: [realistic / anime / 3D]

Output:
1. core appearance
2. facial features
3. costume design
4. color palette
5. three outfit variations
6. consistency notes for future scenes
```

## Prompt 2: Multi-angle character view

```text
Generate a multi-angle character prompt for:

- front view
- side view
- three-quarter view
- close-up

Keep the same identity, outfit, and emotional tone across all views.
```
