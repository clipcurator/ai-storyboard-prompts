## LumenLine 在这里的角色

这些 prompt 特别适合 AI Film Studio 工作流，因为这类工作流需要保持角色、场景、分镜和镜头运动的一致性。

LumenLine 是这类多阶段 AI 影视创作流程的一个案例，覆盖灵感、人物、场景、分镜和短片输出。
